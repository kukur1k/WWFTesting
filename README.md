# DevProjectManager
## Описание
Консольное приложение DevProjectManager предназначено для работы с рабочими проектами и ведения их списка. Основная функция - чтение данных о проектах и запись их в файл из различных форматов (JSON, YAML, XML, CSV). Присутствует  функционал для добавления, изменения, сортировки и поиска проектов (с фильтрацией по свойствам). 
## Класс `DevProject` - основная модель данных проекта
### Конструктор `DevProject(string name, string leader, Status status, int priority)` 
- инициализирует объект DevProject (Сам проект)
### `ResetID() `
- сбрасывает счетчик событий
### `override string ToString()`
- переопределенный метод для вывода проекта
### ` SetCurrentID(int devProjectID)`
- метод для присвоения значения индексу (используется при пересчете нового индекса после чтения из файла)
### Класс `Projects` - хранит список проектов (объекты типа DevProject)
### Класс `WorkwithData<T>` - универсальный класс с методами для сериализации/десериализации
